# flutter_vless_platform_interface

A common platform interface for the [`flutter_vless`](https://pub.dev/packages/flutter_vless) plugin.

This package defines the platform interface that platform implementations (like `flutter_vless_android` and `flutter_vless_ios`) must implement.

## Usage

This package is not intended to be used directly by end-users. Instead, use the main [`flutter_vless`](https://pub.dev/packages/flutter_vless) package.

## Platform Implementations

- Android: [`flutter_vless_android`](https://pub.dev/packages/flutter_vless_android)
- iOS: Implemented directly in the main package (for now).
