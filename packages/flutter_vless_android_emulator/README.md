# flutter_vless_android_emulator

This package contains the x86_64 binaries for the `flutter_vless` plugin, which are required for running on most Android emulators.

## Usage

Add this package to your `pubspec.yaml` alongside `flutter_vless` if you need emulator support.

```yaml
dependencies:
  flutter_vless: ^x.y.z
  flutter_vless_android_emulator: ^x.y.z
```
